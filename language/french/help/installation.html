Installation

1. Préparatifs

Décompressez le contenu du fichier compressé dans le répertoire Modules dans l'arborescence de votre site. Un dossier appelé pages sera créé dans le répertoire modules et un autre dans le répertoire uploads

Le dossier créé dans le répertoire uploads sera utilisé pour la récupération des images et fichiers html que vous pourriez être amené à utiliser dans la création de vos pages.

2. Installation du module

Ce module s'installe simplement comme tous les modules xoops. Il vous suffit d'aller dans le menu Administration, puis dans le menu contextuel effectuez un clic gauche sur l'item Modules.

La partie inférieure de cette page vous affiche les modules non installés dans laquelle il vous suffit de cliquer sur l'icone du module Pages pour débuter l'installation.
