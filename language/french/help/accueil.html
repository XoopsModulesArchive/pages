Ce module que l'on pourrait qualifier de module de sites "Vitrine" ou de couteau suisse des modules est un module de publication de pages indépendantes ou non.

Site vitrine

Si vous souhaitez réaliser un site pour être présent sur internet et présenter votre activité (commerce, profession libérale, etc...) ce module associé à un module contact et un module de plan de site (pour le référencement) est tout à fait à même de remplir cette tâche.

Couteau suisse

Il vous permet par exemple d'intégrer dans votre site :

  • une page complète sur votre gestion des données privées de vos visiteurs ou membres
  • le règlement de votre site
  • la présentation des activités de votre entreprise. Dans la réalisation d'un site vitrine c'est très souvent le module principal sur lequel repose le site.

Avantages

  • Les données publiées dans ces pages sont accessibles par le système de recherche de xoops.
  • simplicité d'utilisation : le module supporte le langage html, mais pourvu de plusieurs éditeurs wisiwig, on écrit et on publie, c'est tout
  • possibilité que chaque page soit automatiquement ajoutée comme sous-lien dans le bloc menu principal
  • permet de publier tout ce qui ne rentre pas dans les autres modules xoops
  • intégration du hack titre des pages pour que chaque page ait un titre différent
  • inclusion du moteur metagen : la fonction Metagen permet d'améliorer le référencement des pages qui utilisent ce procédé en alimentant la balise Meta-Keywords à partir du contenu de celles-ci
  • support des flux rss pour vos pages et/ou possibilité de les insérer sur un autre site avec un script javascript
